# RenewalEngineAI Website

This directory contains the source code for the RenewalEngineAI website.

## Structure

```
website/
├── src/                 # Source code
│   ├── components/      # Reusable components
│   ├── pages/           # Page components
│   ├── styles/          # CSS/Tailwind styles
│   └── lib/             # Utility functions
├── public/              # Static assets
└── next.config.js       # Next.js configuration
```

## Pages to Create

Based on MEMORY.md structure:
- `/` (homepage) - Hero with two paths: consulting & courses
- `/consulting` - What we build, case studies, book a call
- `/courses` - Course catalog, curriculum previews
- `/ai-retention-course` - Flagship course sales page
- `/resources` - Blog, guides, free tools
- `/about` - Our story, team
- `/contact` - Calendly embed for booking
- `/roi-calculator` - Interactive ROI calculator
