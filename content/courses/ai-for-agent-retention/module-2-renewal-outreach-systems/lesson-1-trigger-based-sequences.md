# Trigger-Based Renewal Sequences

A good renewal sequence is not a blast. It is a set of small, well-timed touches fired by specific events on each policy. This lesson walks you through the exact trigger schedule that works for independent agents and how to wire it up.

## The schedule that works
After testing this with dozens of agents, the rhythm below produces the best results on personal lines policies. Adjust the intervals slightly for commercial accounts.

**Day X minus 75:** Light-touch relationship email. Not about the renewal. About them. A market update, a local tip, a quick check-in. This resets the relationship before the renewal conversation starts.

**Day X minus 60:** Value reminder. What did this policy do for them in the last year? Any claims paid, any coverage adjustments, any savings. This is the lesson that turns a commodity into a partnership.

**Day X minus 45:** Review offer. "I'm going to take a fresh look at your coverage before it renews. Anything changed in the last year I should know about?" This produces the cross-sell conversations.

**Day X minus 30:** The renewal itself arrives. You deliver it with a short, personal note.

**Day X minus 14:** Silent check-in. For clients who haven't responded to any of the earlier touches, a short "just making sure you got everything" nudge.

**Day X minus 7:** Escalation for the top 20 percent of the book. A phone call, a text, a direct human touch. This is where you stop automating and pick up the phone.

## What each trigger fires
Behind the schedule sits a simple event system. The trigger is always the X-date minus some number of days. The action is always to send a specific message through a specific channel (email, SMS, or a task on your calendar).

The tool that fires the event does not need to be fancy. Most agents use ConvertKit, Mailchimp, or ActiveCampaign. The important part is that each message is generated at the moment of sending, not written in advance and stored as a static template. We'll get to the generation mechanics in the next two lessons.

## The one rule you cannot break
Every touch must be sent by a real email address that belongs to a real person. No "noreply@agency.com." If a client replies, they must reach you. That single rule separates a retention sequence that feels like spam from one that feels like service.

**Do this today:** Pick one of your next ten renewals. Write down, for that policy, exactly which date corresponds to X minus 75, X minus 60, X minus 45, X minus 30, X minus 14, and X minus 7. These are the six calendar anchors for your first test sequence.
